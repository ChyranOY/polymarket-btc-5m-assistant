  <name>Task 1: Create reconciliation domain module with tests</name>
  <files>src/domain/reconciliation.js, test/domain/reconciliation.test.js</files>
  <action>
    Create src/domain/reconciliation.js — pure function, no I/O:

    Export SYNC_STATUS enum: { IN_SYNC: 'in_sync', CHECKING: 'checking', DISCREPANCY: 'discrepancy' }

    Export reconcilePositions(localPositions, clobPositions, opts):
      Parameters:
        - localPositions: Array of { tokenID, qty, side } from OrderManager/LiveExecutor
        - clobPositions: Array of { tokenID, qty, side } from CLOB API (client.getTrades() computed)
        - opts.graceWindowMs: number (default 10000) — ignore orders placed within this window to avoid timing false positives
        - opts.qtyTolerance: number (default 0) — allow small qty differences (e.g., rounding)

      Algorithm:
        1. Build maps: localMap[tokenID] = { qty, side }, clobMap[tokenID] = { qty, side }
        2. For each local position:
           - If not in CLOB: discrepancy 'LOCAL_ONLY' (we think we have it, CLOB disagrees)
           - If qty differs beyond tolerance: discrepancy 'QTY_MISMATCH'
           - If side differs: discrepancy 'SIDE_MISMATCH'
        3. For each CLOB position not in local: discrepancy 'CLOB_ONLY' (CLOB has it, we don't track it)
        4. Return { status: SYNC_STATUS, discrepancies: Array<{type, tokenID, local, clob, detail}> }

      Returns:
        - No discrepancies: { status: 'in_sync', discrepancies: [] }
        - Any discrepancy: { status: 'discrepancy', discrepancies: [{type, tokenID, local, clob, detail}] }

    Create test/domain/reconciliation.test.js:
    - Test matching positions -> in_sync
    - Test local has position CLOB doesn't -> LOCAL_ONLY discrepancy
    - Test CLOB has position local doesn't -> CLOB_ONLY discrepancy
    - Test qty mismatch -> QTY_MISMATCH discrepancy
    - Test empty local + empty CLOB -> in_sync
    - Test multiple discrepancies reported correctly
    - Test graceWindowMs filters out recent orders (pass recent orders with timestamps)
    - Test qtyTolerance allows small differences
  </action>
  <verify>npm test -- --grep "reconciliation" passes all tests</verify>
  <done>Pure reconciliation module detects all discrepancy types between local and CLOB positions.</done>
